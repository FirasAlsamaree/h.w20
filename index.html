For my Rice UX/UI boot camp project on Energy.gov,
I tested the website with users to find and fix any issues.
After identifying challenges, I got inspiration from other sites
to improve the user experience. I simplified the site's structure
through wireframing for a cleaner look. The process included
creating prototypes and doing more testing for refinement.
In the end, I summed it up in a short case study, showing the
changes made to enhance the user experience on Energy.gov.
